Release Notes:
==============

1. Add support for Windows 7 and Server 2008 R2.
2. Windows 7 and Server 2008 R2 driveres are available via Windows Update
3. Drivers for all other Microsoft Operating Systems will no longer be available via Windows Update.
4. Fix problem with performing immediate reads with a full RX buffer.
